# Ligaro - M&A Integration Services Website

A modern, minimalist website for Ligaro, an AI-first company specializing in tactical M&A integration services.

## Overview

Ligaro helps enterprises execute the complex operational work required during mergers and acquisitions. We combine deep industry expertise with cutting-edge AI to deliver efficient, accurate, and strategic integration support.

## Website Features

- **Modern Minimalist Design**: Clean, professional aesthetic that conveys sophistication and expertise
- **Multi-Page Navigation**: Home, About, Services, and Contact pages
- **Parallax Scrolling**: Smooth, engaging visual effects throughout the site
- **Responsive Design**: Fully responsive for desktop, tablet, and mobile devices
- **Service Organization**: Services grouped into three strategic pillars:
  - Technical Integration
  - Organizational Integration
  - Process & Change Management
- **Contact Form**: Professional form for meeting requests
- **SEO-Friendly**: Semantic HTML and proper meta tags

## Deployment

This website is designed to be deployed as a static site on Azure Static Web Apps.

### Deployment Steps

1. Create an Azure Static Web Apps resource
2. Connect to this GitHub repository
3. Configure the build:
   - Build preset: None
   - App location: `/`
   - Output location: `/`
4. Deploy

## Customization

### Branding
- Update logo text in navigation (currently "Ligaro")
- Modify primary color: `#0066cc` appears throughout CSS
- Update favicon in HTML head

### Content
- Edit text in respective HTML files
- Update service descriptions in services.html
- Modify contact email: `hello@ligaro.com`

### Images
- Replace placeholder hero images with your own
- Update image URLs in HTML files or CSS
